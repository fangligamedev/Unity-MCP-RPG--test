# 9 Kings Prototype Spec Tech: UI Flow

## 1. 主界面
- `New Run`
- `Load Run`
- `Quit`

## 2. 主战局界面
- 顶栏：年份、生命、金币、敌王、阶段
- 中央：5x5 棋盘
- 底部：手牌区
- 左侧：井
- 右侧：Plot 详情与战斗摘要

## 3. 弹窗流程
- Royal Council
- Blessing reveal / resolve
- Diplomat War / Peace
- Merchant
- Tower Expand
- Reward Draft
- Final Battle Alert

## 4. GM 面板
- 顶部状态摘要
- 文本命令输入
- 最近命令日志
- 快捷按钮：暂停、恢复、重开、改年份、改金币、改生命、强制事件、强制胜负
